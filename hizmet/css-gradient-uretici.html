CSS Kodu

CSS Gradient Üretici, web tasarımında kullanabileceğiniz renk geçişlerini hızlı bir şekilde oluşturmanıza olanak tanır. Hem lineer (linear) hem de radyal (radial) gradientler üretebilir ve seçtiğiniz renklerle görsel açıdan şık geçişler elde edebilirsiniz.

Nasıl Kullanılır?

İlk renk ve ikinci renk kutularını kullanarak renkleri seçin.

Gradient türünü (lineer veya radyal) seçin.

Gradientin yönünü belirlemek için kaydırıcıyı kullanın.

“Gradient Üret” butonuna tıklayarak sonucu görsel ve CSS kodu olarak görüntüleyin.
