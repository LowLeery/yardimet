HTML Escape / Unescape Aracı

Bu araç, HTML karakterlerini güvenli bir şekilde dönüştürmenize yardımcı olur. Metinleri HTML formatına dönüştürebilir veya HTML kodlarını normal metne çevirebilirsiniz.

Örnek Dönüşümler:

HTML Escape Örneği:

Giriş: 
Merhaba Dünya

Çıkış: <div>Merhaba Dünya</div>

HTML Unescape Örneği:

Giriş: <p>Bu bir paragraf</p>
Çıkış: 
Bu bir paragraf

HTML Dönüşüm İpuçları:

  • HTML Escape, özel karakterleri HTML varlıklarına dönüştürür
  • HTML Unescape, HTML varlıklarını orijinal karakterlere dönüştürür
  • XSS saldırılarını önlemek için kullanıcı girdilerini escape edin
  • HTML kodlarını düzenlerken unescape işlemi kullanın
  • Dönüşüm sonuçlarını kontrol edin ve gerekirse düzenleyin
Karakter HTML Varlığı Açıklama
< &lt; Küçüktür işareti
> &gt; Büyüktür işareti
& &amp; Ve işareti
" &quot; Çift tırnak
' &#39; Tek tırnak
